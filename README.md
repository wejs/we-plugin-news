# We.js News plugin

Plugin with highlight and widget support Edit
Add topics

## Instalation:
```
we i we-plugin-news
```

## API

### Lista notícias:
```
get /news
```

### Criar notícia:
```
post /news
```

### Carregar dados de 1 notícia:
```
get /news/:id
```

### Atualizar 1 notícia:
```
update|patch /news/:id
```

### Deletar 1 notícia:
```
delete /news/:id
```
